# TFT 시즌 16 라이즈 해금 도우미 (TFT Set 16 Ryze Unlock Helper)

TFT 시즌 16에서 **라이즈(Ryze)**를 해금하기 위한 조건인 **"서로 다른 4개 지역 시너지 활성화"**를 가장 적은 비용과 변경으로 달성할 수 있도록 도와주는 웹 애플리케이션입니다.

![Screenshot](https://via.placeholder.com/800x400?text=TFT+Ryze+Helper+Preview)
*(여기에 실제 스크린샷을 추가하면 더 좋습니다)*

## ✨ 주요 기능

- **스마트 솔루션 탐색**: 현재 내 보드 상태에서 **라이즈를 포함하여** 4개 지역 시너지를 켜는 최적의 방법을 찾아줍니다.
    - **유닛 추가**: 남은 레벨 자리가 있다면 누구를 추가해야 할지 알려줍니다.
    - **유닛 교체**: 자리가 부족하다면 최소한의 유닛을 교체하여 목표를 달성하는 방법을 제안합니다.
- **상징(Emblem) 시스템 지원**: 데마시아, 녹서스 등 보유한 지역 상징을 입력하면 이를 반영하여 계산합니다. (상징이 있으면 해금이 훨씬 쉬워집니다!)
- **편리한 UI**:
    - 게임 내 보드와 유사한 **2x5 그리드** 인터페이스.
    - 챔피언 이름, 비용, 특성별 **필터링 및 검색**.
    - 중복 챔피언 추가 방지 및 시각적 알림.
- **한국어 완벽 지원**: 모든 챔피언, 특성, 지역 이름이 한국어로 제공됩니다.

## 🚀 사용 방법

1. **내 보드 구성**: 우측 리스트에서 현재 내가 사용 중인 챔피언들을 클릭하여 보드에 추가합니다.
2. **레벨 설정**: 현재 나의 레벨(최대 유닛 수)을 입력합니다.
3. **상징 입력**: 만약 지역 상징(아이템)을 가지고 있다면 "보유 상징" 메뉴에서 추가합니다.
4. **솔루션 찾기**: `해금 솔루션 찾기` 버튼을 클릭합니다.
5. **결과 적용**: 제안된 조합 중 마음에 드는 것을 선택하여 `적용하기`를 누르면 보드에 반영됩니다.

## 🛠 기술 스택

- **Friendend**: HTML5, CSS3, JavaScript (Vanilla)
- **Deployment**: GitHub Pages (Static Hosting)

## 📦 설치 및 실행

이 프로젝트는 정적 웹사이트이므로 별도의 설치가 필요 없습니다.
`index.html` 파일을 웹 브라우저로 열기만 하면 바로 실행됩니다.

**GitHub Pages 배포 시:**
1. 이 저장소의 모든 파일을 GitHub Repository에 업로드합니다.
2. Repository Settings -> Pages로 이동합니다.
3. Source를 `Deploy from a branch`로 설정하고 `main` 브랜치를 선택합니다.
4. 잠시 후 생성되는 URL로 접속하여 사용합니다.
